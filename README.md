# Graph Gists
Neo Technology Field Graph Gists

Instructions
===

1. Create a new directory for your graph gist inside gists.
2. Create an images folder inside of your own graph gist folder for any images.
3. Use http://www.apcjones.com/arrows/# for model images if possible
4. Follow the graph_gist_template.adoc in the gists folder.
